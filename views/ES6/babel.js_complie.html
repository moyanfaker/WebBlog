# 方式2---编译JS文件

1.安装Node.js、初始化项目

npm init -y

2.安装babel-cli

npm i @babel/core@babel/cli@babel/preset-env -D

npm i @babel/polyfill-S

3.在package.json中添加执行脚本

"scripts":{
"build":"babel src -d dest"
}

4.添加.babelrc配置文件
